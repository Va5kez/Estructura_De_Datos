========================================================================
    CONSOLE APPLICATION : Colas Project Overview
========================================================================
Este proyecto es un ejemplo de como poder hacer uso de Colas.

Este proyecto hace el uso de colas en lo que es el ingreso y almacenamiento
de chars.

Podemos encontrar 4 funciones para este programa:
1-verCola : Muestra la cola que hemos creado
2-popCola : Saca la char de la cola
3-pushCola : Ingresa una char a la cola
4-menu : Menu en el que escogemos entre las opciones de la cola
