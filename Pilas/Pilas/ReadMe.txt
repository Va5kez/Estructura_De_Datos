========================================================================
    CONSOLE APPLICATION : Pilas Project Overview
========================================================================

Este proyecto es un ejemplo de como poder hacer uso de las Pilas.

Este proyecto hace el uso de las Pilas en lo que es el ingreso y almacenamiento
de chars.

Podemos encontrar 4 funciones para este programa:
1-verPila : Muestra la cola que hemos creado
2-popPila : Saca la char de la Pila
3-pushPila : Ingresa una char a la Pila
4-menu : Menu en el que escogemos entre las opciones de la Pila
